# To-Do List App

A simple and interactive To-Do List application built using **HTML**, **CSS**, and **JavaScript**. This project was created to practice **DOM manipulation**, **event handling**, and **Local Storage** by building a functional task management application that saves tasks even after the browser is refreshed.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Save tasks using Local Storage
* Tasks persist after page refresh
* Dynamic UI updates with JavaScript
* Clean and responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation
* Browser Local Storage

## What I Learned

Through this project, I practiced:

* Creating interactive web applications with JavaScript
* Manipulating DOM elements dynamically
* Handling user input and events
* Storing and retrieving data with Local Storage
* Building a responsive and user-friendly interface
* Organizing front-end code for better maintainability

## How It Works

Users can add tasks to the list, mark them as completed, and remove tasks when finished. All task data is stored in the browser's Local Storage, allowing the list to remain available even after closing or refreshing the page.
